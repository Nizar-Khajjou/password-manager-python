Password Manager:

A GUI password manager built with Python and Tkinter.

Generates secure passwords and stores credentials locally in JSON format.
Includes error handling for missing or corrupt data files.

Features:
- Generate strong random passwords
- Save website, email, and password to a local JSON file
- Search for saved credentials by website name
- Auto-copies generated password to clipboard

How to run:
1. Install dependencies: pip install pyperclip
2. Run main.py
